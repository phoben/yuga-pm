# 规则：布局设计

**优先级：** 高 (HIGH)
**类别：** 布局设计

## 描述

布局设计决定原型的整体框架和视觉结构。良好的布局使信息层次清晰、用户操作流畅，是原型设计从「能用」到「好用」的关键。

## 布局模式选择

### 常用布局模式

| 布局类型 | 结构特点 | 适用场景 |
|---------|---------|---------|
| 顶部导航布局 | 顶部导航 + 内容区 | 网站、营销页 |
| 侧边栏布局 | 侧边导航 + 内容区 | 后台管理系统 |
| 顶部+侧边布局 | 顶部栏 + 侧边导航 + 内容 | 复杂管理系统 |
| 单页布局 | 滚动浏览全部内容 | 落地页、展示页 |
| 分屏布局 | 左右/上下分屏 | 对比、详情列表 |
| 卡片网格布局 | 网格排列卡片 | 仪表盘、商品展示 |

### 布局选择决策

**按产品类型选择：**

```
后台管理系统
├── 功能少于7个 → 顶部导航布局
├── 功能7-15个 → 侧边栏布局
└── 功能超过15个 → 顶部+侧边布局

移动端应用
├── 内容为主 → 单页滚动布局
├── 功能为主 → 底部导航布局
└── 多标签内容 → 顶部标签布局

营销落地页
├── 单一转化目标 → 单页布局
└── 多个产品介绍 → 分区块布局
```

### 响应式断点

| 断点名称 | 宽度范围 | 布局调整 |
|---------|---------|---------|
| 移动端 | < 768px | 单列、底部导航、折叠菜单 |
| 平板 | 768px - 1024px | 双列、侧边栏可折叠 |
| 桌面端 | 1024px - 1440px | 多列、固定侧边栏 |
| 大屏幕 | > 1440px | 内容居中、增加边距 |

## 栅格系统

### 栅格基础

**标准栅格配置：**

| 设备类型 | 列数 | 间距 | 边距 |
|---------|-----|------|------|
| 移动端 | 4列 | 16px | 16px |
| 平板 | 8列 | 24px | 24px |
| 桌面端 | 12列 | 24px | 32px |

**栅格使用原则：**
- 内容块跨越整数列
- 相关内容使用相邻列
- 重要内容占据更多列
- 保持跨设备的栅格继承关系

### 页面区域划分

**经典管理系统布局：**

```
┌────────────────────────────────────────────┐
│                 顶部栏 (56px)               │
├──────────┬─────────────────────────────────┤
│          │                                 │
│  侧边栏   │          内容区                 │
│ (200px)  │       (min-height: calc)        │
│          │                                 │
│          │                                 │

```

**内容区内部划分：**

```
┌─────────────────────────────────────────────┐
│  页面标题区 (padding: 16-24px)              │
├─────────────────────────────────────────────┤
│  筛选/搜索区 (可选)                         │
├─────────────────────────────────────────────┤
│                                             │

│  分页/底部操作 (可选)                       │

```

## 间距与比例

### 间距系统

**基础间距单位：8px**

| 级别 | 数值 | 使用场景 |
|-----|------|---------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 同组元素间距 |
| md | 16px | 区块内间距 |
| lg | 24px | 区块间间距 |
| xl | 32px | 模块间间距 |
| xxl | 48px | 大区块分隔 |

### 尺寸比例

**黄金比例应用：**
- 主内容区 : 侧边栏 = 2:1 或 3:1
- 内容宽度 : 留白 = 1:0.38（内边距）
- 卡片宽高比 = 16:9 或 4:3

**安全尺寸：**

| 元素 | 最小尺寸 | 推荐尺寸 |
|-----|---------|---------|
| 按钮 | 80×32px | 100-120×36px |
| 输入框 | 200×32px | 根据内容 |
| 卡片 | 280×200px | 根据内容 |
| 图标 | 16×16px | 24×24px |

## 常见错误

- 布局过于拥挤，缺少留白
- 导航层级过多，用户迷失
- 忽略响应式，移动端体验差
- 页面元素对齐不一致
- 比例失调，主次不分
- 固定高度导致内容溢出
