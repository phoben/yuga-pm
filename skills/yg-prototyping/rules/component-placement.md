# 规则：组件放置

**优先级：** 高 (HIGH)
**类别：** 组件放置

## 描述

组件放置是将提取的UI元素合理地安排在页面布局中。正确的组件选择和放置使原型更加专业、易用，并能准确传达需求意图。

## 组件选型指南

### 基础组件映射

| 需求描述 | 推荐组件 | 替代方案 |
|---------|---------|---------|
| 输入文本 | 文本输入框 | 文本域（多行） |
| 选择选项 | 下拉选择器 | 单选按钮组（≤4个选项） |
| 多选 | 复选框组 | 多选下拉 |
| 日期选择 | 日期选择器 | 日期范围选择器 |
| 开关状态 | 开关组件 | 复选框 |
| 触发操作 | 按钮 | 链接 |
| 展示状态 | 标签/徽章 | 图标+文字 |
| 数据列表 | 表格 | 卡片列表 |
| 分步流程 | 步骤条 | 标签页 |

### 按功能选型

**数据展示类：**
| 数据类型 | 组件选择 |
|---------|---------|
| 结构化数据 | 表格 |
| 卡片式内容 | 卡片网格 |
| 时间线/流程 | 时间线 |
| 统计数据 | 数字卡片 |
| 比例数据 | 图表 |

**数据录入类：**
| 录入类型 | 组件选择 |
|---------|---------|
| 短文本 | 文本输入框 |
| 长文本 | 文本域 |
| 单选 | 单选按钮/下拉选择 |
| 多选 | 复选框组/多选下拉 |
| 日期时间 | 日期/时间选择器 |
| 文件上传 | 上传组件 |

## 组件放置规则

### 位置原则

**F型阅读模式：**
```
┌─────────────────────────────────┐
│ ← 视线起点                       │
│   ↓ 主要内容区                   │
│                                 │
│          次要内容区 →            │
│                                 │
└─────────────────────────────────┘
```

**放置优先级：**
1. 左上角：最重要的信息/操作
2. 顶部/左侧：核心功能
3. 右侧/底部：辅助功能
4. 右下角：最次要信息

### 操作按钮放置

**主操作按钮：**
- 位置：页面右上角或表单底部
- 样式：填充色、醒目
- 数量：每区块最多1个

**次操作按钮：**
- 位置：主操作左侧或下方
- 样式：描边或文字按钮
- 数量：不超过3个

**按钮组排列：**
```
表单底部：  [取消] [提交]
工具栏：    [批量删除] [导出]     [搜索]
卡片操作：  [查看] [编辑] [删除]
```

### 表单组件放置

**字段排列规则：**
- 单列布局：从上到下，标签在左或上
- 相关字段：相邻放置
- 必填字段：在前或标注
- 长字段：单独一行

**表单分组：**
```
分组1: 基本信息
├── 姓名（必填）
├── 性别
└── 年龄

分组2: 联系方式
├── 手机号（必填）
└── 邮箱

[取消] [保存]
```

### 列表组件放置

**表格列顺序：**
1. 选择列（如有多选）
2. 标识列（名称/编号）
3. 属性列（状态/分类）
4. 数值列（金额/数量）
5. 时间列
6. 操作列

**操作列放置：**
```
操作项 ≤ 3个：直接展示
┌──────────────────────────────┐
│ [查看] [编辑] [删除]         │
└──────────────────────────────┘

操作项 > 3个：更多折叠
┌──────────────────────────────┐
│ [查看] [编辑] •••            │
│              ├─ 删除         │
│              ├─ 导出         │
│              └─ 审核         │
└──────────────────────────────┘
```

## 组件状态设计

### 状态类型

| 状态 | 触发条件 | 视觉表现 |
|-----|---------|---------|
| 默认 | 初始状态 | 标准样式 |
| 悬停 | 鼠标悬停 | 背景变深/边框高亮 |
| 聚焦 | 键盘聚焦 | 外发光/边框高亮 |
| 激活 | 点击/选中 | 强调色填充 |
| 禁用 | 不可操作 | 灰色/透明度降低 |
| 加载 | 处理中 | 加载动画 |
| 错误 | 验证失败 | 红色边框/错误提示 |
| 成功 | 验证通过 | 绿色边框/成功图标 |

### 状态表示方法

**原型中的状态标注：**
```
按钮状态变体：
┌──────────────────────────────┐
│ [正常]  [悬停]  [禁用]       │
│                              │
│ → 主按钮                     │
│ → 次按钮                     │
│ → 文字按钮                   │
└──────────────────────────────┘

输入框状态变体：
┌──────────────────────────────┐
│ [正常]    │ 错误提示文字      │
│ [聚焦]    │ 错误提示文字      │
│ [错误] ╳  │ 错误提示文字      │
└──────────────────────────────┘
```

## 组件组合模式

### 搜索筛选组合

```
简单搜索：
┌────────────────────────────────┐
│ [搜索输入框        ] [搜索]   │
└────────────────────────────────┘

高级筛选：
┌────────────────────────────────┐
│ 筛选条件 ▼ │ 条件值 ▼ │ [+]   │
│ 筛选条件 ▼ │ 条件值 ▼ │ [-]   │
│            [重置] [搜索]       │
└────────────────────────────────┘
```

### 信息展示组合

```
详情卡片：
┌────────────────────────────────┐
│  标题              [操作按钮]  │
│  ────────────────────────────  │
│  字段1：值1                    │
│  字段2：值2                    │
│  ...                           │
└────────────────────────────────┘

统计卡片：
┌────────────────────────────────┐
│  📊 指标名称                   │
│  1,234                         │
│  ↑ 12% 较上周                  │
└────────────────────────────────┘
```

## 常见错误

- 组件类型选择不当（如用输入框展示只读数据）
- 按钮过多，主次不分
- 表单字段排列混乱
- 操作按钮位置不一致
- 忽略组件状态变体
- 组件间距不一致
- 对齐方式混乱
