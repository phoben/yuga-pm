# 活字格帮助

## 适用场景

- 查询活字格功能使用方法
- 解决活字格开发问题
- 了解活字格最佳实践
- 获取活字格API文档

## 使用方式

```bash
# 基本查询
/yg-forguncy-help 关键词

# 示例
/yg-forguncy-help 表格
/yg-forguncy-help 数据绑定
/yg-forguncy-help 页面跳转
```

## 执行流程

```
1. 接收关键词
   └── 用户提供查询关键词

2. 调用搜索脚本
   └── 执行 ai_search.py

3. 查询知识库
   └── 调用活字格官方API

4. 返回结果
   └── 整理并返回相关文档
```

## 脚本文件

- `${CLAUDE_PLUGIN_ROOT}/script/ai_search.py`

## 功能说明

本技能通过调用活字格官方知识库API，为用户提供：

| 功能 | 说明 |
|-----|------|
| 功能查询 | 查询活字格功能使用方法 |
| API文档 | 查询活字格开发API |
| 最佳实践 | 查询活字格使用建议 |
| 问题解决 | 查找常见问题解决方案 |

## 返回内容

查询结果包含：

- 功能说明
- 使用步骤
- 相关API
- 注意事项
- 相关链接

## 搜索技巧

| 技巧 | 示例 |
|-----|------|
| 使用具体功能名 | `表格数据绑定` |
| 包含操作动词 | `如何创建页面` |
| 包含版本号 | `V10新功能` |

## 注意事项

1. 关键词尽量具体，避免过于宽泛
2. 可以使用中英文混合搜索
3. 如果结果不理想，尝试换用同义词
4. 复杂问题建议拆分为多个关键词搜索
