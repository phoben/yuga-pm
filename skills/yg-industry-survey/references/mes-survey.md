# MES 行业知识库

## 一、行业概述

### 1.1 定义与范围

MES（Manufacturing Execution System，制造执行系统）是连接企业规划层（ERP/PLM）与底层控制层（PLC/SCADA）的枢纽级系统，实现生产过程的数字化、透明化管理。

**覆盖范围**：
- 生产计划与排程
- 生产执行与派工
- 质量管理与追溯
- 设备管理与维护
- 数据采集与分析

**演进趋势**：MES 正向 MOM（Manufacturing Operations Management，制造运营管理）演进，从单一执行点（Point MES）转向模块化、协同化的 I-MES/MOM 框架。

### 1.2 核心特征

| 特征 | 说明 |
|------|------|
| 承上启下 | 连接 ERP 计划层与车间控制层 |
| 实时协同 | 生产资源实时调度与协同 |
| 全程追溯 | 从原料到成品的完整追溯链 |
| 数据驱动 | IT/OT 融合，实时决策支持 |

### 1.3 市场格局

- **国际厂商**：Siemens, Rockwell, SAP ME, AVEVA, Honeywell
- **国内厂商**：宝信软件、中控技术、赛意信息、鼎捷软件
- **市场趋势**：工业 4.0、数字化工厂、IT/OT 融合、AI 质检

## 二、核心模块

### 2.1 生产计划与执行

**功能说明**：
- **计划管理**：生产计划接收、转订单、订单排程
- **派工管理**：生产派工、委外派工、进度追踪
- **执行控制**：工位作业、工序流转、完工确认

**核心流程**：
```
ERP计划 → 生产计划 → 转订单 → 派工单 → 车间执行 → 完工确认 → 入库
```

**关键指标**：
| 指标名称 | 计算方式 | 说明 |
|----------|----------|------|
| 计划达成率 | 实际产量 / 计划产量 × 100% | 衡量计划执行 |
| 订单准时率 | 准时完成订单 / 总订单 × 100% | 衡量交付能力 |
| 派工完成率 | 已完成派工 / 总派工 × 100% | 衡量执行进度 |

### 2.2 质量管理

**功能说明**：
- **质检管理**：首检、巡检、终检、入库检
- **追溯管理**：批次追溯、工艺 BOM 差异分析
- **异常管理**：异常发起、处理跟踪、根因分析

**核心流程**：
```
生产完成 → 质检申请 → 质检执行 → 结果判定 → 合格入库/不合格处理
```

**关键指标**：
| 指标名称 | 计算方式 | 说明 |
|----------|----------|------|
| 一次合格率 | 一次合格数 / 送检数 × 100% | 衡量质量水平 |
| 质检覆盖率 | 已检派工 / 应检派工 × 100% | 衡量质检覆盖 |
| 异常处理率 | 已处理异常 / 总异常 × 100% | 衡量响应效率 |

### 2.3 物料管理

**功能说明**：
- **领料管理**：领料单、补料单、退料单
- **物料追溯**：条码集成、批次追踪
- **废料管理**：废料登记、废料分析

**流转凭证**：
| 凭证类型 | 用途 | 触发场景 |
|----------|------|----------|
| 物料登记单 | 物料入库登记 | 物料到达 |
| 领料单 | 生产领料 | 派工开始 |
| 补料单 | 追加领料 | 物料不足 |
| 退料单 | 剩余退库 | 派工结束 |
| 废料单 | 废料登记 | 质检不合格 |

### 2.4 设备管理

**功能说明**：
- **设备台账**：设备档案、设备状态
- **维护管理**：保养计划、维修工单、备件管理
- **数据采集**：设备状态、运行参数、告警信息

**核心流程**：
```
设备监控 → 异常告警 → 维修工单 → 备件领用 → 维修确认 → 恢复运行
```

**关键指标**：
| 指标名称 | 计算方式 | 说明 |
|----------|----------|------|
| 设备稼动率 | 实际运行时间 / 计划运行时间 × 100% | 衡量设备利用 |
| MTBF | 平均故障间隔时间 | 衡量设备可靠性 |
| MTTR | 平均修复时间 | 衡量维护效率 |

## 三、数据模型

### 3.1 核心实体

**生产模型要素**：

| 实体类型 | 实体名称 | 说明 |
|----------|----------|------|
| 基础要素 | 设备 | 生产设备档案 |
| 基础要素 | 工位 | 生产作业位置 |
| 基础要素 | 产线 | 生产流水线 |
| 基础要素 | 车间 | 生产组织单元 |
| 基础要素 | 班组 | 人员组织单元 |
| 逻辑要素 | 工序 | 加工原子动作 |
| 逻辑要素 | 工艺流程 | 工序编排路径 |
| 逻辑要素 | 生产流程 | 产品加工路径 |

**业务实体**：

| 实体名称 | 说明 | 核心字段 |
|----------|------|----------|
| 生产计划 | ERP 下发的计划 | 计划号、产品、数量、日期 |
| 生产订单 | 执行单元 | 订单号、产品、数量、状态 |
| 派工单 | 作业指令 | 派工号、订单、工序、工位、数量 |
| 质检单 | 质量检验 | 质检号、派工、检验项、结果 |
| 异常单 | 异常记录 | 异常号、类型、级别、状态 |

### 3.2 实体关系

```
生产计划 --1:N--> 生产订单
生产订单 --1:N--> 派工单
派工单 --1:1--> 质检单
派工单 --1:N--> 物料消耗
设备 --1:N--> 维修工单
工艺流程 --1:N--> 工序
```

## 四、业务流程

### 4.1 端到端流程

**计划到入库**：
```
ERP计划接收 → 计划分解 → 订单生成 → 排程派工 → 物料准备 → 生产执行 → 质检确认 → 入库
```

**流程说明**：
| 阶段 | 涉及角色 | 关键动作 | 输出物 |
|------|----------|----------|--------|
| 计划接收 | 计划员 | 接收 ERP 计划 | 生产计划 |
| 订单生成 | 计划员 | 转订单操作 | 生产订单 |
| 排程派工 | 调度员 | 派工分配 | 派工单 |
| 物料准备 | 仓管员 | 领料出库 | 领料单 |
| 生产执行 | 操作工 | 作业报工 | 进度记录 |
| 质检确认 | 质检员 | 质量检验 | 质检单 |
| 入库确认 | 仓管员 | 成品入库 | 入库单 |

### 4.2 关键场景

| 场景 | 触发条件 | 处理流程 | 注意事项 |
|------|----------|----------|----------|
| 计划变更 | ERP 计划调整 | 评估影响 → 调整排程 → 通知现场 | 已开工订单处理 |
| 质量异常 | 质检不合格 | 异常登记 → 原因分析 → 处置决策 | 追溯影响范围 |
| 设备故障 | 设备告警 | 停机 → 报修 → 维修 → 恢复 | 订单进度影响 |
| 物料短缺 | 缺料预警 | 补料申请 → 审批 → 发料 | 避免停工待料 |

## 五、集成与扩展

### 5.1 常见集成

| 集成对象 | 集成方式 | 数据流向 | 说明 |
|----------|----------|----------|------|
| ERP | 接口/API | 双向 | 计划下发、完工上报 |
| WMS | 接口/条码 | 双向 | 物料领用、成品入库 |
| PLM | 接口 | 单向 | 工艺路线、图纸文档 |
| SCADA/PLC | 协议采集 | 单向 | 设备状态、生产参数 |
| CAD/CAM | 文件共享 | 单向 | 技术资料、工艺图纸 |

### 5.2 集成逻辑

**条码集成**：
- WMS 物料入库生成条码
- 生产领料扫码确认
- 实现物料全程追溯

**研发集成**：
- CAD/CAM 技术资料共享
- 工艺图纸实时同步
- 确保生产使用最新版本

**设备集成**：
- 设备状态实时采集
- 保养维修数据驱动备件需求
- 实现预防性维护

## 六、最佳实践

### 6.1 实施建议

| 建议 | 原因 | 适用场景 |
|------|------|----------|
| 统一术语体系 | 消除跨部门沟通壁垒 | 项目启动前 |
| 先数据采集后优化 | 数据驱动决策基础 | 分阶段实施 |
| 工艺 BOM 差异分析 | 精准定位质量波动源头 | 质量管理 |
| 异常闭环管理 | 实现秒级响应 | 生产现场 |

### 6.2 常见问题

| 问题 | 解决方案 | 预防措施 |
|------|----------|----------|
| 数据采集不完整 | 自动采集 + 补录机制 | 设备联网优先 |
| 与 ERP 数据不一致 | 统一数据口径、定时对账 | 接口校验机制 |
| 用户抵触录入 | 简化操作、移动端支持 | 用户参与设计 |
| 追溯链条断裂 | 强制扫码、流程卡点 | 流程闭环设计 |

### 6.3 调研问题清单

**生产模式**：
- 生产类型（离散/流程/混合）？
- 订单特点（多品种小批量/少品种大批量）？
- 生产组织方式（车间、产线、工位）？

**计划与排程**：
- 计划来源（ERP/MRP）？
- 排程维度（车间/产线/工位）？
- 计划变更频率？

**质量管理**：
- 质检环节有哪些？
- 追溯要求（批次/单品）？
- 不合格处理流程？

**设备管理**：
- 设备联网情况？
- 设备品牌/协议？
- 维护模式（预防/事后）？

**集成需求**：
- 上游系统（ERP/PLM）？
- 下游系统（WMS/SCADA）？
- 数据同步频率？
