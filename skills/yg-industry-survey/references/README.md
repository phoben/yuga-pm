# 行业知识库

本目录存放行业调研沉淀的知识文档。

## 文档规范

详见 [SKILL.md](../SKILL.md) 「知识库规范」章节。

### 命名规范

```
{行业关键词}-survey.md      # 行业调研文档
{系统类型}-design.md        # 系统设计参考
_template.md               # 知识库模板
```

### 章节结构

| 章节 | 必填 | 说明 |
|------|:----:|------|
| Frontmatter | ✓ | 元数据 |
| 一、行业概述 | ✓ | 定义、特征、市场格局 |
| 二、核心模块 | ✓ | 功能模块、核心流程、关键指标 |
| 三、数据模型 | ○ | 核心实体、实体关系 |
| 四、业务流程 | ✓ | 端到端流程、关键场景 |
| 五、集成与扩展 | ○ | 常见集成、扩展点 |
| 六、最佳实践 | ✓ | 实施建议、常见问题、调研问题清单 |
| 七、参考资料 | ✓ | 引用来源 |
| 变更记录 | ✓ | 版本变更历史 |

> ✓ 必填 ○ 选填

## 使用说明

1. 新增知识库时，复制 `_template.md` 作为起点
2. 按规范填写内容
3. 更新本文件的「现有知识库」表格
4. 更新 [SKILL.md](../SKILL.md) 的「知识库引用」表格
